Purpose and Function
- This library is meant to provide project-agnostic logic and data structures.
- The library is subject to change without notice.
- Inheriting projects may not work with all versions of the library.



Usage
- When a project uses functions in this library, make sure that the folder containing the compiled JAR file has a subdirectory named "lib" that contains the Wheeler.jar file.
- When importing a NetBeans project that uses this library, make sure the library can be found at the location "C:\Program Files\Wheeler\lib\Wheeler.jar"
